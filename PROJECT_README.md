# Fresh Flow Full Project

This file is the full project overview without modifying the existing `README.md`.

## Quick Start

1. Generate sample data: `python scripts/generate_synthetic_data.py --output data/Inventory_Management`.
2. Run pipeline: `python scripts/run_full_pipeline.py --data-dir data/Inventory_Management --output-dir reports`.
3. Optional dashboard: `streamlit run app_streamlit.py`.
